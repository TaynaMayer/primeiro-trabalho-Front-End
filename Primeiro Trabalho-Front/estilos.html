Usando CSS

O CSS pode ser adicionado aos documentos HTML de 3 maneiras:

Inline - usando o atributo dentro de elementos HTMLstyle Interno - usando um elemento na seção"style" "head"

Externo - usando um elemento para vincular a um arquivo CSS externo "link"

Inline CSS

Um CSS inline é usado para aplicar um estilo único a um único elemento HTML.

Um CSS inline usa o atributo de um elemento HTML.style

O exemplo a seguir define a cor do texto do elemento para azul, e a cor do texto do elemento para vermelho:"h1"



Exemplo "h1 style="color:blue;">A Blue Heading"/h1" "p style="color:red;">A red paragraph."/p"

CSS interno Um CSS interno é usado para definir um estilo para uma única página HTML.

Um CSS interno é definido na seção de uma página HTML, dentro de um elemento."head""style"

CSS externo

Uma folha de estilo externa é usada para definir o estilo para muitas páginas HTML.

Para usar uma folha de estilo externa, adicione um link a ela na seção de cada página HTML:"head"

Cores, fontes e tamanhos do CSS

Aqui, vamos demonstrar algumas propriedades CSS comumente usadas. Você aprenderá mais sobre eles mais tarde.

A propriedade CSS define a cor do texto a ser usada.color

A propriedade CSS define a fonte a ser usada.font-family

A propriedade CSS define o tamanho do texto a ser usado.font-size

A propriedade CSS define uma borda em torno de um elemento HTML.border Exemplo:

p { border: 2px solid powderblue; }

A propriedade CSS define um preenchimento (espaço) entre o texto e a borda.padding Exemplo:

p { border: 2px solid powderblue;

padding: 30px;

}

A propriedade CSS define uma margem (espaço) fora da fronteira.margin

Exemplo:

p { border: 2px solid powderblue;

margin: 50px;

}

Link para CSS externo

As folhas de estilo externa podem ser referenciadas com uma URL completa ou com um caminho relativo à página web atual.

Exemplo Este exemplo usa uma URL completa para vincular a uma folha de estilo: <

"link rel="stylesheet" href="https://www.w3schools.com/html/styles.css">

Box model

Em CSS, o termo "modelo de caixa" é usado quando se fala de design e layout.

O modelo de caixa CSS é essencialmente uma caixa que envolve cada elemento HTML. Ele consiste em: margens, bordas, preenchimento e o conteúdo real. A imagem abaixo ilustra o modelo da caixa:



Explicação das diferentes partes:

Content - O conteúdo da caixa, onde o texto e as imagens aparecem

Padding - Limpa uma área ao redor do conteúdo. O forro é transparente

Border - Uma borda que contorna o preenchimento e o conteúdo

Margin - Limpa uma área fora da fronteira. A margem é transparente

O modelo de caixa nos permite adicionar uma borda ao redor dos elementos e definir o espaço entre os elementos.
